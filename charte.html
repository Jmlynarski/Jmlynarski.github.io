Charte de Protection des Données

Date de dernière mise à jour : 08/10/2024

La présente Charte de Protection des Données décrit la manière dont nous collectons, utilisons, partageons et protégeons les données des utilisateurs dans le cadre de l’utilisation de notre API TikTok. Nous nous engageons à respecter les lois et réglementations applicables en matière de protection des données, y compris le Règlement Général sur la Protection des Données (RGPD) de l'Union Européenne et d’autres lois sur la confidentialité pertinentes.

1. Responsable du Traitement des Données

Le responsable du traitement des données est CreteilSunset situé au 85 avenue du général de Gaulle. Pour toute question relative à la protection des données, vous pouvez nous contacter à l’adresse suivante : pierre.le-roux.saucy873@simplelogin.com.

2. Données Collectées

Lors de l’utilisation de notre API TikTok, nous pouvons être amenés à collecter les types de données suivants :

  • Données Personnelles : identifiants des utilisateurs TikTok (nom d'utilisateur, identifiant unique), informations de profil public, contenus publiés (vidéos, commentaires, likes).
  • Données Techniques : adresses IP, données de localisation (si autorisé), informations sur l’appareil utilisé (type, version du système d’exploitation).
  • Données d’Utilisation : interactions avec les services TikTok via l'API (durée de session, fréquence des requêtes, API endpoints utilisés).

Nous ne collectons aucune donnée sensible (par ex. : données de santé, croyances religieuses) à travers cette API.

3. Finalités du Traitement des Données

Les données collectées via l'API TikTok seront traitées pour les finalités suivantes :

  • Prestation des Services : permettre l'accès et l'intégration des fonctionnalités TikTok via notre API.
  • Amélioration du Service : analyser les tendances d’utilisation, optimiser les performances de l'API et identifier les bugs ou problèmes techniques.
  • Conformité Légale : répondre aux obligations légales, y compris les demandes des autorités compétentes, et faire valoir nos droits en cas de litige.

4. Base Légale du Traitement

Nous traitons les données collectées sur la base des motifs légaux suivants :

  • Consentement : Lorsque l’utilisateur accorde explicitement son consentement pour la collecte de certaines données (ex. : accès aux données de localisation).
  • Exécution d’un Contrat : Lorsque le traitement est nécessaire pour fournir les services de l’API selon les termes d'utilisation convenus.
  • Intérêts Légitimes : Améliorer et sécuriser notre API et l'expérience utilisateur, sous réserve que ces intérêts ne portent pas atteinte aux droits fondamentaux des utilisateurs.

5. Partage des Données

Les données collectées via l'API TikTok peuvent être partagées avec :

  • Partenaires Tiers : Si nécessaire pour la prestation des services ou l’amélioration de notre API (ex. : hébergeurs de données, services analytiques).
  • Autorités Juridiques : En réponse à des obligations légales ou à des demandes officielles émanant des autorités compétentes.

Nous ne vendons ni ne louons les données personnelles des utilisateurs à des tiers à des fins commerciales.

6. Durée de Conservation des Données

Les données personnelles collectées via l'API TikTok seront conservées uniquement pendant la durée nécessaire aux finalités pour lesquelles elles ont été collectées, à moins qu’une durée de conservation plus longue soit requise ou autorisée par la loi.

  • Données Techniques et d'Utilisation : 6 mois.
  • Données Personnelles : 6 mois ou jusqu'à ce que l'utilisateur révoque son consentement ou demande la suppression de ses données.

7. Sécurité des Données

Nous mettons en œuvre des mesures techniques et organisationnelles appropriées pour protéger les données personnelles contre tout accès non autorisé, toute divulgation, perte ou destruction. Ces mesures incluent le chiffrement des données en transit et au repos, ainsi que des audits réguliers de nos systèmes de sécurité.

8. Droits des Utilisateurs

Conformément à la législation applicable en matière de protection des données, les utilisateurs disposent des droits suivants :

  • Droit d’Accès : Demander une copie de leurs données personnelles traitées via notre API.
  • Droit de Rectification : Demander la correction de données inexactes ou incomplètes.
  • Droit à l’Effacement : Demander la suppression de leurs données personnelles sous certaines conditions.
  • Droit à la Portabilité : Recevoir leurs données dans un format structuré et couramment utilisé pour les transférer à un autre fournisseur.
  • Droit d’Opposition et de Limitation : S’opposer ou limiter certains traitements de leurs données.

Les utilisateurs peuvent exercer ces droits en nous contactant à l’adresse : pierre.le-roux.saucy873@simplelogin.com.

9. Transferts Internationaux de Données

Dans le cadre de l’exploitation de notre API, les données peuvent être transférées en dehors de l’Espace Économique Européen (EEE). Dans ce cas, nous nous engageons à ce que ces transferts respectent les règles de protection des données applicables, en utilisant des mécanismes adéquats tels que les clauses contractuelles types approuvées par la Commission Européenne.

10. Modifications de la Charte

Nous nous réservons le droit de modifier cette Charte de Protection des Données à tout moment. Toute modification sera communiquée aux utilisateurs via notre site web ou par e-mail, selon le cas. Nous encourageons les utilisateurs à consulter régulièrement cette Charte pour se tenir informés de toute mise à jour.

11. Contact

Pour toute question ou préoccupation concernant cette Charte ou le traitement de vos données, veuillez nous contacter à pierre.le-roux.saucy873@simplelogin.com.
